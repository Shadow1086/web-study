# Web学习项目文档

## 📚 项目结构

本项目按照学习阶段和难度分为四个主要模块：

### 1️⃣ 01-HTML基础

**学习内容：** HTML基础标签和属性

- 常见文本标签
- 标签的关系
- 标题标签
- 换行和水平线标签
- 文本格式化标签
- 图像标签和属性
- 超链接标签
- 音频和视频标签
- 字符实体
- HTML属性

**学习目标：** 掌握HTML的基本标签，能够创建简单的网页结构

---

### 2️⃣ 02-HTML进阶

**学习内容：** 复杂的HTML结构和表单

- 列表（无序、有序、定义列表）
- 表格（基本用法、结构标签、合并单元格）
- 表单（输入框、单选框、多选框、下拉菜单、文本域、按钮）
- 布局标签
- 实战案例（个人简历、注册信息）

**学习目标：** 能够使用列表、表格、表单等复杂结构，完成实际项目

---

### 3️⃣ 03-CSS基础

**学习内容：** CSS基础知识和选择器

- CSS体验
- CSS引入方式
- 标签选择器和类选择器
- ID选择器
- 通配符选择器
- 盒模型基础

**学习目标：** 掌握CSS选择器，理解盒模型概念，能够对网页进行基本样式设置

---

### 4️⃣ 04-CSS进阶

**学习内容：** CSS文字控制和复合选择器

- 文字控制属性
- 行高和垂直居中
- 字体设置
- font复合属性
- 文本缩进
- 文本对齐
- 文字修饰线
- 文字颜色
- 复合选择器
- 实战案例（新闻网页）

**学习目标：** 掌握文字排版，使用复合选择器，完成实际网页设计

---

## 🎯 学习建议

1. **按顺序学习**：从01-HTML基础开始，逐步进阶
2. **动手实践**：每个文件都是可以直接在浏览器中打开的HTML文件
3. **对比学习**：在浏览器中打开文件，查看效果，修改代码，观察变化
4. **完成案例**：在02-HTML进阶和04-CSS进阶中有实战案例，要重点学习

---
